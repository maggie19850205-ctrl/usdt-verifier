# GEO Readiness Checker

A simple CLI tool to check if your website is optimized for Generative Engine Optimization (GEO). Tests your site against criteria used by ChatGPT Search, Perplexity AI, Google AI Overview, and Claude.

## Features

- Check meta tags, schema markup, and content structure
- Score your site's GEO readiness (0-100)
- Get actionable improvement suggestions
- Zero dependencies ¡ª single Python script
