## 提交步骤

### 第 1 步：注册/登录 Product Hunt
1. 浏览器打开 https://www.producthunt.com/
2. 点击右上角 "Sign in"
3. 选择 "Continue with Google" 或 "Continue with Twitter"（不要用 Apple）
4. 登录成功后，打开 https://www.producthunt.com/posts/new

### 第 2 步：填写基本信息
| 字段 | 填入内容 |
|------|----------|
| Product Name | `MaoMaoLove` |
| Tagline | `Digital Products Marketplace Powered by USDT TRC-20 — Instant Delivery, No Registration` |
| Description | 复制上面准备好的描述 |
| URL | `https://maomaolove.pages.dev` |
| YouTube | 留空 |

### 第 3 步：上传图片
- **Logo**: 上传截取的 logo 图
- **Gallery images**: 上传 3-4 张截图
- 建议第一张展示首页，第二张展示产品页，第三张展示支付流程

### 第 4 步：选择分类
- **Category**: `Productivity` 或 `Tech`
- **Subcategory**: 选合适的（如 `SaaS`, `Developer Tools`）

### 第 5 步：设置 Launch 日期
- 建议选 **周一至周四**（流量最大），不要选周五到周日
- 选好日期后检查 "Schedule" 确认

### 第 6 步：发布
1. 点击 "Submit" 提交审核
2. 审核通过后，在选定日期自动发布
3. 发布当天打开页面，在评论框粘贴准备好的 **First Comment**
4. **拉人来 upvote** — 发到你的微信/Telegram 群，让大家点 "Upvote"

### 2. AlternativeTo
1. 打开 https://alternativeto.net/ （先注册账号）
2. 搜索相关产品（如 "Gumroad", "Etsy", "Payhip"）
3. 在每个产品页面点 "Add alternative"
4. 填入 MaoMaoLove 信息

### 3. SaaSHub
1. 打开 https://www.saashub.com/submit
2. 注册账号
3. 填入产品信息
4. 注意要绕过 Cloudflare 验证（可能需要用真实浏览器）
